# 勤怠管理アプリ — 要求仕様書

## 概要

中規模（〜100人）の組織向け勤怠管理 Web アプリケーション。
社員の出退勤打刻、勤怠実績の閲覧、有給休暇管理、残業アラートを提供する。

## ロールと権限

| ロール | 権限 |
|--------|------|
| 一般社員 | 自分の打刻、自分の勤怠閲覧、有給申請 |
| 部門長 | 上記 + 自部門社員の勤怠閲覧、有給申請の承認、残業アラート受信（部門社員分） |
| 管理者 | 全社員の勤怠閲覧・編集、社員マスタ管理、カレンダーマスタ管理 |

## ユーザーストーリー

### 認証

- US-01: 社員として、ID/パスワードでログインしたい。自分の勤怠情報にアクセスするために。
- US-02: 社員として、ログアウトしたい。セッションを終了するために。

### 打刻

- US-10: 社員として、出勤打刻をしたい。勤務開始時刻を記録するために。
- US-11: 社員として、退勤打刻をしたい。勤務終了時刻を記録するために。
- US-12: 社員として、1日に複数回の出退勤を記録したい。中抜け勤務に対応するために。
- US-13: 社員として、打刻を修正したい（時刻の手動入力可）。打刻忘れや誤打刻を訂正するために。
  - 承認フローなし。本人が直接修正可能。

### 勤怠一覧

- US-20: 社員として、自分の月次勤怠一覧を確認したい。日ごとの出退勤時刻・勤務時間を確認するために。
- US-21: 部門長として、自部門社員の月次勤怠一覧を確認したい。部下の勤務状況を把握するために。
- US-22: 管理者として、全社員の勤怠一覧を確認・編集したい。勤務実績を管理するために。

### 有給休暇

- US-30: 社員として、有給休暇を申請したい。休暇を取得するために。
- US-31: 部門長として、部下の有給申請を承認/却下したい。休暇取得を管理するために。
- US-32: 社員として、自分の有給残日数を確認したい。取得計画を立てるために。

### 残業アラート

- US-40: 社員として、月の残業が30時間を超えた場合にアラートを受け取りたい。過重労働を防ぐために。
- US-41: 部門長として、部下の残業が30時間を超えた場合にアラートを受け取りたい。部下の健康を守るために。

### 社員マスタ（管理者）

- US-50: 管理者として、社員を登録したい。新入社員の利用を開始するために。
- US-51: 管理者として、社員情報を編集したい。異動や変更を反映するために。
- US-52: 管理者として、社員を削除（無効化）したい。退職者のアカウントを停止するために。

### レポート（MVP 後）

- US-60: 管理者として、月次勤怠レポートを CSV/PDF で出力したい。給与計算や監査に使うために。

## ビジネスルール

### 勤務時間の計算

| 項目 | ルール |
|------|--------|
| 所定労働時間（定時） | 7時間15分/日 |
| 休憩控除 | 1時間（自動控除） |
| 残業判定 | 実働 > 7時間15分 の超過分 |
| 残業割増率 | x1.25 |
| 中抜け対応 | 1日に複数回の出退勤を記録可。各区間の合計が実働時間 |

### 有給休暇

| 項目 | ルール |
|------|--------|
| 年間付与日数 | 一律20日 |
| 繰越 | あり（未消化分を翌年度に繰越） |
| 申請フロー | 社員が申請 → 部門長が承認/却下 |

### 残業アラート

| 項目 | ルール |
|------|--------|
| 閾値 | 月30時間超過 |
| 通知先 | 本人 + 部門長 |

### 休日・祝日

- カレンダーマスタで管理（管理者が設定）
- 祝日・会社独自の休日を登録可能

## マスタデータ

| マスタ | 管理者 | 備考 |
|--------|--------|------|
| 社員マスタ | 管理者が CRUD | 部門への所属を持つ |
| 部門マスタ | 初期データあり（管理画面なし） | シードデータで投入 |
| カレンダーマスタ | 管理者が設定 | 年間の休日・祝日を定義 |

## 非機能要件

| 項目 | 要件 |
|------|------|
| 利用者規模 | 〜100人（中規模） |
| 対応デバイス | PC のみ（レスポンシブ不要） |
| データ保持期間 | 1年 |
| 認証 | ID/パスワード認証 |
| 技術スタック | Java / Spring Boot（Backend）、TypeScript / Next.js（Frontend） |

## MVP スコープ

### MVP に含む

- 認証（ログイン/ログアウト）
- 打刻（出勤/退勤、複数回対応、修正）
- 勤怠一覧（本人/部門長/管理者の各ビュー）
- 有給休暇（申請、承認、残日数表示）
- 残業アラート（30時間超過通知）
- 社員マスタ管理
- カレンダーマスタ管理

### MVP 後（Phase 2）

- 月次勤怠レポート出力（CSV/PDF）

# 画面設計

## 画面一覧

| # | 画面名 | パス | ロール | 説明 |
|---|--------|------|--------|------|
| 1 | ログイン | `/login` | 全員 | ID/PW 入力 |
| 2 | ダッシュボード | `/` | 全員 | 今日の打刻状態・アラート表示 |
| 3 | 打刻 | `/attendance` | 全員 | 出勤/退勤ボタン、当日の打刻履歴 |
| 4 | 勤怠一覧（自分） | `/attendance/monthly` | 全員 | 月次カレンダー形式の勤怠表 |
| 5 | 勤怠一覧（部門） | `/attendance/department` | 部門長 | 部門社員の月次一覧 |
| 6 | 勤怠一覧（全社） | `/admin/attendance` | 管理者 | 全社員の勤怠閲覧・編集 |
| 7 | 有給申請 | `/leave/request` | 全員 | 日付選択・申請 |
| 8 | 有給申請一覧 | `/leave/requests` | 全員/部門長 | 自分の申請 or 部門の申請一覧 |
| 9 | 社員管理 | `/admin/employees` | 管理者 | 社員一覧・登録・編集・無効化 |
| 10 | カレンダー管理 | `/admin/calendar` | 管理者 | 休日の登録・削除 |
| 11 | アラート | `/alerts` | 全員/部門長 | 残業超過アラート一覧 |

## コンポーネント構成

```
src/
├── app/
│   ├── layout.tsx              # 共通レイアウト（Nav）
│   ├── page.tsx                # ダッシュボード
│   ├── login/page.tsx
│   ├── attendance/
│   │   ├── page.tsx            # 打刻画面
│   │   └── monthly/page.tsx    # 月次一覧
│   ├── leave/
│   │   ├── request/page.tsx    # 有給申請
│   │   └── requests/page.tsx   # 申請一覧
│   ├── admin/
│   │   ├── employees/page.tsx  # 社員管理
│   │   ├── attendance/page.tsx # 全社勤怠
│   │   └── calendar/page.tsx   # カレンダー管理
│   └── alerts/page.tsx         # アラート一覧
├── components/
│   ├── ui/                     # 汎用UIコンポーネント
│   ├── attendance/             # 打刻関連
│   ├── leave/                  # 有給関連
│   └── admin/                  # 管理者画面
└── lib/
    ├── api-client.ts           # fetch ラッパー
    └── types.ts                # API レスポンス型
```

# 勤怠管理アプリ — フォローアップ Q&A

## 追加質問

### ロール・権限

[Question] 部門長の権限は何ですか？（例: 自部門の社員の勤怠閲覧、打刻修正の確認 etc.）管理者との違いは？
[Answer]自部門の社員の勤怠閲覧

[Question] 管理者は全社員の勤怠を閲覧・編集できる想定ですか？
[Answer]はい

### 有給休暇

[Question] 有給20日は年度ごとにリセットですか？未消化分の繰越はありますか？
[Answer]繰越あり

[Question] 有給申請にはワークフロー（申請→承認）が必要ですか？それとも自己申告で即反映？
[Answer]部門長へワークフローあり

### 残業アラート

[Question] 残業30時間超過のアラートは誰に表示しますか？（本人のみ、本人＋部門長、本人＋管理者）
[Answer]本人＋部門長

### 部門

[Question] 社員は「部門」に所属する想定ですか？部門マスタの管理も必要ですか？
[Answer]所属する　部門マスタの管理はなし
